📨 Alertas Telegram [telegram_alerts]

Envía notificaciones a uno o más chats de Telegram cuando ocurren eventos relevantes (conexión/desconexión de radios, Brew up/down, etc.).
CONFIGURACIÓN — 4 PASOS
  1. En Telegram, abre @BotFather, envía /newbot y sigue las instrucciones. Copia el token que te da.
  2. Pega el token abajo y pulsa Verificar — verás el @usuario de tu bot.
  3. Abre un chat con tu nuevo bot (o añádelo a un grupo) y envíale cualquier mensaje, p. ej. /start.
  4. Pulsa "Detectar Chat ID", añade el chat a los destinatarios y guarda. Usa "Enviar test" para confirmar.
TOKEN DEL BOT
El token de @BotFather tiene el aspecto 123456789:AAExampleTokenString. Se almacena enmascarado y no se muestra completo.
DESTINATARIOS (CHAT IDS)
Cada alerta se envía a todos los destinatarios. ID positivo = chat privado; ID negativo = grupo o canal.
